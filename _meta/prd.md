Product: Bounded Alignment Protocol

### Problem
In low-information markets, negotiation leads to exploitation, regret, and unstable agreements.

### Solution
A two-step alignment protocol that allows agreements only when both parties are already compatible within bounded flexibility.

### Target Markets
- Hiring (mid/senior roles)
- Freelance/consulting
- Art sales
- Partnerships

### Non-target Markets
- Commodities
- Auctions
- Enterprise negotiated deals

### User Flow
1. Initiator configures range + spread
2. Candidate submits first range
3. System evaluates overlap
4. If no overlap → feasibility check
5. If feasible → direction revealed
6. Second bid
7. Final decision

### Success Metrics
- Offer acceptance rate
- Post-deal satisfaction
- Reduction in negotiation rounds
- Reduction in offer-stage drop-offs

### Risks
- Reduced deal volume
- Poor initial calibration
- User confusion about constraints

### Positioning
"No bad deals. Only aligned ones."
